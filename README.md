# Smart Power Load Forecasting and Grid Stability Analyzer

This project is a web-based engineering tool that predicts electrical load using numerical interpolation techniques and analyzes grid stability conditions.

## Features

• CSV dataset upload  
• Load forecasting using interpolation  
• Newton Forward Difference Method  
• Newton Backward Difference Method  
• Lagrange Interpolation  
• Grid stability indicator  
• Load profile visualization  
• 24-hour load forecasting dashboard  
• Actual vs predicted load comparison  

## Technologies

- HTML
- CSS
- JavaScript
- Chart.js

## Dataset Format

CSV format:

hour,load

8,120  
9,140  
10,165  
11,190  

## How It Works

1. Upload historical load data (CSV)
2. Select interpolation method
3. Predict load at a specific time
4. View grid stability status
5. Visualize load profile on the dashboard
